****************************************
                  CONTROLS
****************************************
LMB - Interact / Shoot
RMB - Zoom / Switch to FIRST PERSON VIEW

Num1 - Use Item in Inventory #1
Num2 - Use Item in Inventory #2
Num3 - Use Item in Inventory #3

R - Reload

A - Unzoom / Switch to FRONT VIEW
D - Unzoom / Switch to SIDE VIEW

Space - Pause Game

****************************************
             SPECIFICATIONS
****************************************
Boulder Blocker	: Diagonal Blocker + Circular Hitbox
Wisp Blocker	: Vertical Blocker + Triangular Hitbox
Shadow Blocker	: Horizontal Blocker + Rectangular Hitbox
Cage Blocker	: Special Vertical Blocker + Rectangular Hitbox

Health Boost [Activated]	: Heals player 1-3 hearts
Instant Kill [Activated]	: Defeats all present enemies

Damage Boost [Acquired]	: Deals double damage
Piercing Shot [Acquired]	: Pierces through all entities
Infinity Ammo [Acquired]	: Does not decrement ammo
